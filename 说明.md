第一步：创建本地项目文件
在本地创建一个文件夹，并在其中创建以下四个文件。
1. cjpm.toml
[package]
cjc-version = "0.53.10"
name = "cj_demo"
version = "0.0.1"
2. src/main.cj
package main
func add(a: Int, b: Int): Int {
    a + b
}
main() {
    println("Hello Cangjie")
}
3. tests/test_main.cj
package test
import main.*
@Test
func testAdd() {
    if (add(1, 1) != 2) {
        throw Exception("Test Failed")
    }
    println("test_add passed")
}
@Test
func testAddWithZero() {
    if (add(5, 0) != 5) {
        throw Exception("Test Failed")
    }
    println("test_add_with_zero passed")
}
4. .gitlab-ci.yml (GitCode 流水线配置文件)
stages:
  - test
job_test:
  stage: test
  image: ubuntu:latest
  before_script:
    - apt-get update && apt-get install -y wget
  script:
    - wget https://gitee.com/OpenHarmony/cangjie/releases/download/v0.53.10/cangjie_0.53.10_linux-x86_64.tar.gz
    - tar -xzf cangjie_0.53.10_linux-x86_64.tar.gz
    - export PATH=$PATH:$PWD/cangjie/bin
    - cjpm test
第二步：推送到 GitCode
在 GitCode 上新建一个空仓库。
在本地项目文件夹下打开终端，依次执行以下命令：
git init
git add .
git commit -m "init"
git branch -M main
git remote add origin https://gitcode.net/你的用户名/你的仓库名.git
git push -u origin main
*(注意：请将命令中的仓库地址替换为你实际的 GitCode 仓库地址)*
第三步：查看结果（截图素材）
登录 GitCode，进入你的仓库。
点击左侧菜单 “CI/CD” -> “流水线”。
等待约 1-2 分钟，刷新页面，看到状态变为 “成功”（绿色勾选）。
点击该流水线记录进入详情页，查看 Job 日志，滚动到底部确认能看到 test_add passed 和 test_add_with_zero passed 字样。
保存截图。
